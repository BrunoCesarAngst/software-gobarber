# software-gobarber
Um software de agendamento de beleza

Iniciando o projeto
`mkdir software-gobarber`
`cd software-gobarber`
`yarn init -y`
`yarn add express`

Conseito de padronização de código
`mkdir src`
`cd src`
`touch app.js`
`touch server.js`
`touch routes.js`
